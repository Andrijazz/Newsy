=====

Dependencies
============

- MongoDb cpp driver
    - Official docs: http://docs.mongodb.org/ecosystem/drivers/cpp/
    - Repo: https://github.com/mongodb/mongo-cxx-driver

- Boost C++ libraries
    - apt-get install libboost-all-dev

- Flex: 
    - apt-get install flex
